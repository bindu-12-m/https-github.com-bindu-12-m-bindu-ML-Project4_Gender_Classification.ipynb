# Gender Classification using Facial Geometry

## 📌 Project Overview
This project aims to classify gender (Male/Female) using facial geometric features. 
Instead of using raw images, the model uses numerical facial ratios, making it computationally efficient.

---

## 🎯 Objective
To build a machine learning model that predicts gender using:
- Facial geometric ratios
- PCA for dimensionality reduction
- Weighted K-Nearest Neighbors (KNN) for classification

---

## 📂 Dataset
- Contains facial geometry ratios (35 features)
- Target variable: Gender (0 = Male, 1 = Female)
- Source: Kaggle / Research dataset

---

## ⚙️ Methodology
1. Data Preprocessing  
2. Feature Scaling (StandardScaler)  
3. Dimensionality Reduction using PCA (8 components)  
4. Model Training using Weighted KNN  
5. Evaluation using Accuracy and Confusion Matrix  

---

## 📊 Results
- Model achieved good accuracy in gender classification  
- PCA reduced dimensionality effectively  
- Weighted KNN improved classification performance
